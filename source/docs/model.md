---
title: 模型
---
Hawa Code 支持多种模型，支持原生模型同时因为适配 OpenAI 协议，所以兼容 OpenAI 协议的模型都可以支持。

| 模型 | 支持状态 | Base URL |
|------|----------|----------|
| Kimi | 支持（推荐） | https://api.moonshot.cn |
| Deepseek | 支持 | https://api.deepseek.com |
| Gemini | 支持（Beta） | https://generativelanguage.googleapis.com |
| OpenAI | 支持 | https://api.openai.com |
| Anthropic | 支持 | https://api.anthropic.com |
