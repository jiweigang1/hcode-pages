title: Models
---
Hawa Code supports multiple models, supporting native models while also being compatible with any models that support the OpenAI protocol due to OpenAI protocol adaptation.

| Model | Support Status | Base URL |
|------|----------|----------|
| Kimi | Supported (Recommended) | https://api.moonshot.cn |
| Deepseek | Supported | https://api.deepseek.com |
| Gemini | Supported (Beta) | https://generativelanguage.googleapis.com |
| OpenAI | Supported | https://api.openai.com |
| Anthropic | Supported | https://api.anthropic.com |
